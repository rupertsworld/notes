# Telepath Offsite
#events #telepath

We settled on the following plan for the next week:

- **Obsidian plugin:** build a plugin that's focused on information retrieval, and solves this problem by allowing users to spontaneously create (and optionally save) live "smart notes" that synthesize and retrieve information for a topic or query. 
- **Agent UI:** start work on a persistent workspace UI for OpenClaw, NanoClaw, Claude Code, and other agents. This gives the agents the ability to show artifacts of various type to the user, in a magnetic canvas space, with multiple workspaces. The agent can read/write to this canvas. 

Our strategy is to determine which path is most viable (or if both are). Part of this will also involve getting more information about these communities, and what the core influencers in these spaces are boosting as buzz is a central priority.

## Other decisions made 

- We are moving away from Notion and towards a markdown/GitHub-based approach (perhaps using Stash)
- We want to use hot-reloading to be able to make UI using specs. We are going to write a blog post about this. 
- We will have three tiers of project: personal experiments (do what you want), team experiments (light spec-driven, no approvals), and products (formal spec-driven with PRs and approval from Josh)

## Other promising ideas 

- **Agent context hub:** (AKA agentic filesystem), a desktop app/CLI and service where you can connect all your skills, MCP servers, and other data sources like files. It then syncs this across all your agents. You can also have profiles for different agents, for example only sharing some data with an agent you want to live publicly, or with a coding agent, or one that you use at work. 

## Other notes & comments

- Each of these parts is complimentary to a Telepath Computer approach: a generative UI, and deep personalisation. 
- A standalone app that doesn't tie into a community or pre-existing product is off the table for now, as we are specifically looking to generate buzz and ride a wave. 
- Stephen feels like the "Agent UI" path is more compelling for his pitch & storytelling than the Obsidian approach
