# Agent background execution

Execute commands with agents in the background and track running jobs. Agents themselves get a tool to dispatch background work — they return a summary or report, full messages are posted.

If in a git repo, option to spin up a disposable worktree — replicate the repo, do work on a branch, push it. Cleanup or retention TBD.

## Telepath

Part of the broader open agent stack. Exercises orchestration and agent infrastructure — areas Telepath may move into. Could become a product, an open source package, or part of the company OS.
