# Agent context hub

An agentic filesystem / desktop app+CLI where you connect all your skills, MCP servers, and data sources (files, mail, calendar, etc.). It syncs this across all your agents.

Could start as a CLI — connect mail & calendar, and it adds skills to your agent automatically.

Profiles let you scope what's shared: e.g. only some data with a public agent, different data for a coding agent vs a personal one vs a work one.

## Telepath

Complements the Telepath Computer approach — deep personalisation layer. From the Sedona offsite: each piece (generative UI, context hub, orchestration) is part of the same picture.
