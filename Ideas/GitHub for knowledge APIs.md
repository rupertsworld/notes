# GitHub for Knowledge APIs & Datasets

A platform where people publish, version, and discover knowledge APIs and datasets purpose-built for agents. Like GitHub but for structured data and queryable endpoints rather than code.

Agents need reliable, structured access to domain knowledge — not just raw web search. This would be the registry and hosting layer for that: versioned datasets, API specs, usage tracking, forking, community contributions.

Think: npm/GitHub but the consumers are agents, not developers.
