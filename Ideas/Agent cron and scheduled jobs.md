# Agent cron and scheduled jobs

A jobs directory with cron-like scheduling for background agents. Triggers:

- **Time-based** — actual cron schedule
- **File/folder change** — watch-based, more experimental

Agents run in the background on trigger (using [[Agent background execution]]), produce reports or take actions.

## Telepath

Part of an open agent stack that overlaps with Telepath's future product areas. Internal experimentation first, potentially a product or open source package later.
