# Physical agent input devices
#ai #design #interfaces #agents

Alternative physical input modalities for controlling agents beyond keyboard and mouse.

## Approaches

- **Foot pedals** — Mapping pedals to keys or microphone inputs for hands-free agent control.
- **Gamepads** — Using game controllers for agent interaction.
- **Voice** — Microphone-based input for natural language commands.
