#ai #design #interfaces #computing #generative

## Schema-based UI

This is related to how A2UI works, and how LangView worked. You specify a domain-agnostic schema and send structured data to a front-end that is responsible for rendering it, and sending back appropriate actions.

Some examples of these schema languages in use:

- [Airbnb's server-driven UI system](https://medium.com/airbnb-engineering/a-deep-dive-into-airbnbs-server-driven-ui-system-842244c5f5), they basically run their entire app interface on this
