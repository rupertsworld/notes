# Web styling

- [CSS Grid Lanes](https://webkit.org/blog/17660/introducing-css-grid-lanes/) - native CSS masonry layouts with `display: grid-lanes`
- [Oat](https://oat.ink/) - tiny (6KB CSS, 2.2KB JS) classless component library. Semantic HTML styled out of the box, zero dependencies, dark mode, accessible. Good counterpoint to framework-heavy UI kits.
