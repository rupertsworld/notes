# Push notification services

Services for sending push notifications without building a native app.

## ntfy.sh

https://ntfy.sh/

- HTTP-based pub-sub notifications via simple PUT/POST
- Open source (Apache 2.0 / GPLv2), self-hostable
- iOS, Android, desktop apps
- No signup required - topics act as passwords
- Free tier, paid plans for higher limits

## ActivitySmith

https://activitysmith.com/

- iOS-specific: push notifications + Live Activities
- Handles APNs infrastructure, certs, app store submission
- SDKs: Node.js, Python, Go, PHP, Ruby, CLI
