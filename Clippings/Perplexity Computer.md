# Perplexity Computer
#products #ai

Perplexity launched "Computer" on Feb 25, 2026 — an agentic product that goes beyond their chat/search interface. Users describe an end goal (build a website, write a report, generate a dataset) and Computer breaks it into tasks handled by specialized sub-agents. It auto-selects the best model per task (e.g. image generation, deep research, quick search) and can run autonomously for hours or longer.

Essentially: orchestrated multi-agent workflows with web search as a core capability, packaged for consumers.

## Commentary

This looks like what Claude/ChatGPT can already do today plus cron jobs — neat and useful, but not a new paradigm. The real unlock isn't more capabilities stuffed into chat UIs; it's integration into a user's life and new UX metaphors. Their "computer" is not what I'd consider a "computer." That said, Perplexity needs to keep pushing here — they're selling a product that packages models to do something useful in a user's life, and web search as part of an agent is rapidly becoming table stakes.
