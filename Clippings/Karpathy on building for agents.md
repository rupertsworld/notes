# Karpathy on building for agents
#ai #agents

https://x.com/karpathy/status/2026360908398862478

CLIs are exciting because they are "legacy" technology that AI agents can natively use, combine, and interact with via the terminal toolkit. Ask an agent to install a CLI and it can build dashboards, interfaces, or logic on top of it.

For any product or service, consider:

- Are your docs exportable in markdown?
- Have you written Skills for your product?
- Can your product be used via CLI or MCP?

> "It's 2026. Build. For. Agents."
