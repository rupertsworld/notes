# Hex: Voice → text, paste anywhere
#design #interfaces #products

https://github.com/kitlangton/Hex

Press-and-hold (or double-tap) a global hotkey to record voice; transcribes and pastes into whatever app has focus. Parakeet TDT v3 / FluidAudio or on-device WhisperKit. Apple Silicon only. `brew install --cask kitlangton-hex`. Needs mic + accessibility.
