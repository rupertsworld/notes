# Claude Code's docs map
#ai #agents #web #information

Claude Code uses this 'docs map' which is hosted on their docs website. It's a high-level overview & index of all the documentation that an agent might want to view with links. They structure it so each area has a heading and a link, with nested dot-points of what is in each document.

Article on this by Simon Willison: https://simonwillison.net/2025/Oct/24/claude-code-docs-map/

URL of a map: https://code.claude.com/docs/en/claude_code_docs_map.md

## Example

```
## Document Structure

This map uses a hierarchical structure:

* **##** marks documentation groups (e.g., 'Getting started')
* **###** marks individual documentation pages
* **Nested bullets** show the heading structure within each page
* Each page title links to the full documentation

## Getting started

### [overview](https://code.claude.com/docs/en/overview.md)

* Get started
* What you can do
* Use Claude Code everywhere
* Next steps
  
...
```
