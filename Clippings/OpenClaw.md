## Security

Vignesh (@_vgnsh) open-sourced formal security models (TLA+) for OpenClaw, machine-checked with TLC. Models cover gateway exposure/auth, node run approvals, and misconfiguration scenarios. Not a proof of the full codebase, but machine-checked evidence that intended security rules enforce claimed properties, with counterexample traces for realistic bugs.

- Thread: https://x.com/_vgnsh/status/2016015059521306827
- Formal models: https://github.com/vignesh07/clawdbot-formal-models
- Security docs: https://docs.clawd.bot/gateway/security/formal-verification
