# Chowder: AI-first mobile chat patterns
#ai #design #interfaces #products

https://github.com/newmaterialco/chowder-iOS

Native iOS chat client for OpenClaw that demonstrates AI-first mobile interface patterns. Key design decisions worth studying: real-time streaming via WebSocket, live activity tracking that shows thinking steps and tool execution with haptic feedback, dynamic identity sync from workspace files.

Relevant for Telepath as a reference for how to surface AI agent activity transparently rather than treating it as a black box. The "verbose mode" approach to showing what the agent is doing could inform our own activity indicators.
