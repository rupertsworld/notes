# ActivitySmith: Live Activities from any backend
#ai #design #interfaces #products

https://activitysmith.com/

Backend service that lets you send iOS Live Activities and push notifications without building a custom app. Handles APNs infrastructure, certificates, and app store submission. SDKs for Node.js, Python, Go, PHP, Ruby, plus CLI.

Relevant as infrastructure for agent-to-user async communication. Pair with an agent backend to push real-time status updates (thinking, tool execution, completion) to Lock Screen and Dynamic Island without building native iOS code.
