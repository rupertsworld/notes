# Brother HL-L3220CDW

Color laser printer.

## Toner

- **TN-229C** — Cyan (standard yield)
- **TN-229XL C** — Cyan (high yield)
- Same series for other colors: TN-229M, TN-229Y, TN-229BK (and XL variants)

## Troubleshooting

### "Slide the green tab on the drum unit" for all colors

A single empty toner cartridge can trigger this message for all four colors. Check toner levels first — replacing the empty cartridge clears the message.

### Corona wire cleaning

Open front cover → pull out drum/toner assembly → slide the green tab on each drum unit fully left→right→left 3-4 times → **return tab to home (leftmost) position** or the error persists. One tab per color (CMYK).

### Getting more life from a low cartridge

Remove cartridge, rock gently side to side 5-6 times to redistribute toner. Temporary fix, good for a few dozen pages.
