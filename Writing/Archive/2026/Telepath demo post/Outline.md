## Breaking it down

Dynamic interfaces & the death of the app:

- In a personal computer today, developers define the applications and interfaces that a user can use. While this convenient, it's also heavily restrictive/disempowering. And applications are still completely pre-determined, so users are required to do all the manual work to operate them.
- With Telepath, there are no apps. Instead, the interface fluidly composes based on the task at hand.
- This simultaneously eliminates a lot of manual effort, while also turning a computer from a set of discrete possible actions one can take within a given set of apps, to a fluid, continuous action space where the user can have their wish be the user's command.

Sensemaking:

- This computer is deeply personal. It builds and maintains an understanding of you as a user, and uses that for input in every decision it makes.
- This "sensemaking" capability is also critical to execute a dynamic interface right. Just as software developers need to understand their target user to build an app, in order to create a good interface on the fly your computer needs to know a lot about you.
- This computer is able to know what to withhold, as well as what to present.

Situational awareness:

- Computers now are inflexible in terms of their input forms. In a world where software is pre-programmed and deterministic, you need to specify the types of input you'll accept in advance. That means for a Desktop app, I need to be at my computer to operate the buttons and menus given to me. For mobile, I need to be poking my finger at my phone. But for the Telepath computer, there is no such restriction.
- In the demo, we hinted at this by focusing on voice input with a mix of spoken and visual feedback. You can imagine that if Stephen approached the computer he would see more on-screen affordances. If he turned away, output would by primarily audio with more detail.
- As a result, the borders between "desktop" and "mobile" blur too. Software doesn't have to exist in one place to be useful, and it doesn't have to be written for any place in particular.
